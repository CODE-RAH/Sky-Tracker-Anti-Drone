# 📝 درباره پروژه

این پروژه یک سیستم بینایی ماشین پیشرفته جهت پایش حریم هوایی و تشخیص اجسام متحرک کوچک (مانند انواع پهپادها، کوادکوپترها و پرندگان) در بستر آسمان است.

نرم‌افزار مجهز به دو مکانیزم عملیاتی قدرتمند است:

1. **اسکن و تشخیص خودکار (Auto Detect)**  
   با آنالیز فریم‌ها، هر نوع شیء متحرکی که با سرعت و ابعاد مشخص در آسمان حرکت کند را فوراً به عنوان هدف شناسایی می‌کند.

2. **قفل روی هدف و ردیابی پایدار (Target Locking)**  
   به کاربر اجازه می‌دهد به صورت دستی یا خودکار روی یک سوژه قفل کرده و با دقت بسیار بالا، مسیر حرکت آن را با کراس‌هیر (Crosshair) تعقیب کند.

# ⚡ ویژگی‌های فنی و معماری پردازش تصویر

- **الگوریتم تفریق پس‌زمینه MOG2**  
  استفاده از مدل ترکیب گوسی برای حذف عناصر ثابت یا نوسانی پس‌زمینه و استخراج دقیق ماسک حرکتی اهداف.

- **فیلترهای مورفولوژیکی ترکیبی (Opening & Closing)**  
  اعمال عملیات ریاضی روی ماسک تصویر با کرنل‌های بیضوی جهت حذف نویزهای سنسور و یکپارچه‌سازی نواحی متحرک.

- **ردیاب پیشرفته CSRT Tracker**  
  استفاده از `TrackerCSRT` برای ردیابی پایدار اهداف کوچک هوایی حتی در شرایط تغییر نور یا تغییر فرم جسم.

- **فیلترینگ هندسی هوشمند**  
  بررسی نسبت ابعاد (`W/H`) و مساحت کانتورهای حرکتی جهت تشخیص دقیق‌تر پهپادها و جلوگیری از خطاهای محیطی.

# 🚀 راهنمای جامع دانلود، نصب و راه‌اندازی (قدم به قدم)

## 📥 مرحله اول: دانلود فایل پروژه از گیت‌هاب

به دو روش می‌توانید پروژه را دانلود کنید:

### روش اول — دانلود ZIP

روی دکمه سبز رنگ **Code** کلیک کرده و گزینه **Download ZIP** را انتخاب کنید.

سپس فایل ZIP را از حالت فشرده خارج نمایید.

### روش دوم — دانلود مستقیم فایل

روی فایل پایتون پروژه کلیک کرده و سپس گزینه **Download raw file** را بزنید تا فایل مستقیماً دانلود شود.

---

## 🧩 مرحله دوم: نصب پایتون

اگر Python روی سیستم شما نصب نیست:

1. وارد سایت رسمی Python شوید.
2. آخرین نسخه را دانلود و نصب نمایید.
3. هنگام نصب، حتماً گزینه زیر را فعال کنید:

```text
Add Python to PATH
```

---

# 📦 مرحله سوم: نصب کتابخانه‌ها و پیش‌نیازها

1. وارد پوشه پروژه شوید.
2. کلید `Shift` را نگه دارید و راست کلیک کنید.
3. گزینه **Open PowerShell window here** یا **Open in Terminal** را انتخاب نمایید.

سپس دستور زیر را اجرا کنید:

```bash
pip install opencv-contrib-python numpy
```

> ⚠️ توجه:  
> برای استفاده از ردیاب `CSRT` باید حتماً پکیج `opencv-contrib-python` نصب شود و نسخه معمولی OpenCV کافی نیست.
